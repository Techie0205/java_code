# Java Training Programs

## 📌 Description

This repository contains basic Java programs like Even-Odd, Prime, Palindrome, Calculator, etc.

## ⚙️ Requirements

* Java JDK installed
* VS Code or any IDE

## ▶️ How to Run

1. Open terminal in project folder
2. Compile:
   javac FileName.java
3. Run:
   java FileName

## 📂 Files Included

* EvenOdd.java
* PrimeCheck.java
* Palindrome.java
* Calculator.java
* and many more...
